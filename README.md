# Credit Fraud & Spend Intelligence
**What behavioral and demographic patterns predict high spending and do fraudulent transactions mimic those same patterns or have clearly different signatures? This is an end-to-end machine learning project that answers that question. It analyzes transaction behavior to model spending patterns and detect fraud, with feature engineering, Prophet, XGBoost, LightGBM, TabNet deep learning, Optuna optimization, and SHAP-based interpretability. Dataset from https://www.kaggle.com/datasets/priyamchoksi/credit-card-transactions-dataset/data**
